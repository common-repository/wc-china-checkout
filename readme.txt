WooCommerce中国本地结算,三级联动收货地址,物流查询,订单信息推送到海关。

== Description ==
一款专门为中国人打造的WooCommerce结算和物流通知插件，包含中国式收货地址（省，市，城市三级联动），账单字段自定义，拖拽字段，物流通知功能，支持全国绝大部分物流公司。界面简洁，性能优越，适配国人的使用习惯。订单信息推送到海关，更有其他功能可定制。企业用户请购买企业版：<a href="https://www.wpweixin.net/product/2105.html" target="_blank">WC China Checkout 物流通知插件</a>

= Features = 
* 支持WooCommerce账单地址新增，删除，修改(可拖拽式自定义表单字段)
* 可完全关闭WooCommerce账单地址，适合虚拟商品
* 支持中国式收货地址，省，市/地区，城市三级联动
* 支持订单物流通知功能，兼容全国绝大部分物流公司(高级扩展，需单独购买)
* 支持账单地址手机号码验证(高级扩展，需单独购买)
* 订单微信通知（开发中，下版本更新）
